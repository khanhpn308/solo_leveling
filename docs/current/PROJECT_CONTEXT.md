# Project Context

## Product Goal

This project is a local IELTS Academic self-study dashboard built for a long-running personal study campaign.

The target user profile is:

- current level around `B1`
- strongest area: Listening
- weaker area: Reading, especially vocabulary range and long-sentence comprehension
- long-term target: IELTS Academic `7.0-7.5`

## Product Shape

The dashboard is designed as a game-style system interface that makes long-term study feel more rewarding and easier to sustain.

Core ideas:

- visible progression
- short feedback loops
- streak and quest pressure
- skill ranks from `F` to `S`
- XP from completed tasks, not from time spent

## Main Feature Surfaces

- dashboard / player profile
- skill progress
- quest board
- daily quests
- weekly missions
- boss battles
- badge wall
- check-ins
- error log
- writing tracker
- speaking tracker
- mock test tracker

## Current Direction

- the frontend is already in an English-first state
- backend schema migration Waves `A-E` are complete
- campaign-scoped mutable state is now the live model
- documentation is organized for low-noise context loading
