# IELTS Quest Dashboard - Project Context

## User profile
The user is a 24-year-old Vietnamese learner who recently graduated from university and is currently studying a master's program in Vietnam. The user may later study a master's degree abroad, so the roadmap is focused on IELTS Academic.

## IELTS goal
Target: IELTS Academic 7.0–7.5 within 18 months.

Target skill bands:
- Listening: 7.5–8.0
- Reading: 7.5–8.0
- Speaking: 6.5
- Writing: 6.5–7.0
- Overall: 7.0–7.5

## Current level
Estimated current level: around B1.

Known test history:
- TOEIC Listening: 395
- TOEIC Reading: 270
- TOEIC Speaking: 95
- TOEIC Writing: 120
- Aptis General: Overall B1
- Aptis Listening: B2
- Aptis Reading: B1
- Aptis Speaking: B1
- Aptis Writing: B1

## Main weakness
Reading is weak because of:
- limited vocabulary
- difficulty understanding long sentences

## Main strength
Listening is currently the strongest skill.

## Study schedule
The user wants to study:
- 4 main sessions per week
- 2 hours per session
- plus 15–20 minutes of daily mini study

## Dashboard concept
The website should act like a personal RPG system for IELTS learning.

The learner is the player.
Each study task is a quest.
Each skill is a character stat.
Each month has a boss battle.
Each milestone unlocks badges.
Each completed task gives XP.
Each skill has rank F to S.

## Skills
Track these skills:
- Listening
- Reading
- Writing
- Speaking
- Vocabulary
- Collocation
- Grammar

## Rank logic
Ranks:
- F: beginner / inactive
- E: started
- D: foundation
- C: stable beginner-intermediate
- B: IELTS 5.5–6.0 range
- A: IELTS 6.5–7.0 range
- S: IELTS 7.5+ target zone

## Roadmap phases
Phase 1: Month 1–3
- Build foundation
- Use Complete IELTS Band 5.0–6.5
- Use English Grammar in Use
- Use English Vocabulary in Use Upper-Intermediate
- Use English Collocations in Use Intermediate

Phase 2: Month 4–6
- Start serious IELTS format training
- Add IELTS Advantage Reading Skills
- Add IELTS Advantage Writing Skills
- Add IELTS Advantage Speaking and Listening Skills
- Continue Cambridge Grammar for IELTS

Phase 3: Month 7–9
- Move toward IELTS 6.0–6.5
- Use Complete IELTS 6.5–7.5
- Use IELTS Vocabulary for Bands 6.5 and above
- Practice Writing Task 2 and Speaking Part 2/3

Phase 4: Month 10–12
- Start partial test practice
- Use Cambridge IELTS 17 carefully
- Add Cambridge Academic Vocabulary in Use
- Begin English Collocations in Use Advanced

Phase 5: Month 13–18
- Mock tests
- Error correction
- Full Listening and Reading tests
- Weekly Writing correction
- Speaking recording and review
- Aim for IELTS 7.0–7.5

## Key UX requirement
The dashboard must prevent boredom and broken streaks. It should make small progress visible every day.
